# browseupdate 浏升
:arrow_up:低版本浏览器升级提示标准页面

## 项目缘由
在我司项目中，主要系统使用者都是小白，浏览器五花八门，而发现给他们Chrome之类的浏览器官网进行下载他们并不会，而且Chrome还被墙了，所以，我做了这样一个网页：

1. 列出现代主流浏览器，包括国外和国内的
2. 直接提供它们的下载地址，点击图标直接下载到电脑
3. 一些升级提示文案等
4. 如果是IE9以下额外在顶栏显示一个红色的文案

这个网页可以作为同用的模版使用

## 计划  
- [ ] 自动获取浏览器最新版本号，与浏览器官方版本一致  
- [ ] 自动更新下载链接，与浏览器官方版本一致
